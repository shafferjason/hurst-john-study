HOW IT ALL WORKS TOGETHER:

1. User visits index.html
2. JavaScript reads data.js
3. Generates list of lessons automatically
4. User clicks a lesson
5. lesson.html loads
6. JavaScript finds that lesson in data.js
7. Displays all content from data.js
8. Loads audio from /audio/ folder
9. Loads image from /images/ folder
10. Auto-generates Previous/Next buttons
